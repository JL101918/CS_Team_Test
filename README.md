
## Run it 

now to run your little code, do that:
```bash
dotnet run --project CS_Team_Test/CS_Team_Test.csproj
```

## Project creation:

The four main pillars are:

```bash
dotnet new console
dotnet restore
dotnet build
dotnet run
```

Dotnet new console --> project file (.csproj) / starter C# file (program.cs) and a basic folder layout. Its like creating the structure for the code to run.

Dotnet restore --> checks if all C# #libraries are there (and downloads any libraries that are missing).

Dotnet build --> compiler reads files and tries to convert it to clanker language (sorry, binary), if build fails it breaks.

Dotnet run  --> runs it :D
